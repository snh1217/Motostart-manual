# Motostar Manual Hub

## Getting Started
```bash
npm run dev
```

Open http://localhost:3000

## Ops Notes
- If you change Vercel env vars such as `MANUALS_CACHE_BUST` or `MANUALS_VERSION`, you must redeploy to apply them.
- If you use Supabase Storage for PDFs, add your production domain to Storage CORS Allowed Origins.
