Diagnosis Tree Manual Checklist

1) /admin/diagnosis
- Upload valid JSON -> saved
- Upload invalid JSON -> blocked with error message
- Toggle ACTIVE/INACTIVE -> reflected on list

2) /diagnosis
- With active trees -> model select -> tree select -> wizard runs
- With empty DB -> JSON fallback is used

3) Result actions
- "관련 정비사례 보기" opens filtered /cases
- "사례 업로드 초안(CSV/XLSX)" downloads prefilled template
